# 遠端桌面連線

## rdesktop
製作可遠端連線的對話框

1. 將 rdesktop.sh 複製到 /usr/local/bin
2. 將 rdesktop.png 複製到 /usr/share/pixmaps
3. 將 rdesktop.desktop 複製到　~/桌面

## 利用 imagemagick 製作文字圖型

`make_cht_png.sh 文字內容 檔名`

## 利用 imagemagick 轉成圓型圖

`png2round.sh 原圖檔 輸出檔名`
